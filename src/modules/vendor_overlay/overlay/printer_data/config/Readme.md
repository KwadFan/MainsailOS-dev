## How to flash the mcu?

-   Copy over the `config` file from `Firmware-config`

```bash
cp -v ~/printer_data_config/Firmware-config/klipper-config ~/klipper/.config
```

-   Run firmware build process by

```bash
cd ~/klipper
make -j$(nproc)
```
